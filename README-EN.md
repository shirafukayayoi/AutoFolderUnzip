# AutoFolderUnZip

## Description

This script automatically extracts zip files in a folder.  
It's useful when you need to extract a lot of zip files.  

## Usage

1. Install the requirements using the `requirements.txt` file:  
`pip install -r requirements.txt`
2. Run the `AutoFolderUnZip.py` script:  
`python AutoFolderUnZip.py`
3. Enter the path of the folder you want to extract.
4. Place the zip files you want to extract in the specified folder.
5. The script will automatically extract the zip files for you!
